• A variable is a named container for storing data values. It allows you to store data that you can use and modify later in your code.
  • You can declare variables using three keywords:
    1. var
      • function-scoped
      • Can be re-declared and updated
      • Hoisted(can be accessed before declaration, but value will be undefined)
    2. let
      • Block-scoped(only available inside{})
      • Cannot be re-declared in the same scope
      • Hoisted, but cannot access before declaration
    3. const
      • Block-scoped
      • Cannot be changed after it's assigned
      • Must be initialized when declared
  • Variable Naming Rules
    • Can contain letters, numbers, $, or _
    • Cannot start with a number
    • Cannot use reserved words like let, if, for as variable names
  • Primitive Types (Immutable)
    1. String
    2. Number
    3. Boolean
    4. undefined
    5. Null
    6. Symbol
    7. Bigint
